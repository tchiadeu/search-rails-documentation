# Rails API Documentation

Search through Ruby on Rails API documentation directly from Raycast.

## Features

- Fast search across all Rails API documentation
- Browse classes, modules, and methods
- Open documentation in your browser
- Copy documentation URLs to clipboard

## Usage

1. Open Raycast
2. Type "Search Rails Documentation" or use the configured shortcut
3. Start typing to search through Rails classes, modules, and methods
4. Press Enter to open the documentation in your browser
5. Press `Cmd + .` to copy the URL to clipboard

## Data Source

This extension searches the official Ruby on Rails API documentation at [api.rubyonrails.org](https://api.rubyonrails.org/).
